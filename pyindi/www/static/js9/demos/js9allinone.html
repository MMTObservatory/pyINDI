JS9 Demo: the allinone configuration

The js9-allinone.css and js9-allinone.js files can be served via CDN, obviating the need to install JS9 on your web site.

JS9 install files and directories (other than .css and .js files) are not available in this mode, leading to the following limitations:

  • no js9Prefs.json for site-wide configuration
  • no back-end helper will be run
  • no help files will be available (and no Help menu option)
  • no server-side analysis (including proxy load and large file support)

  • 3c58 (Chandra)
